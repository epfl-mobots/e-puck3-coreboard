To create the font or edit it. FontForge was used.

Steps to follow :

1) Create an SVG file of the logo/image you want to integrate to the font
2) Open the FontForge project (sfd file)
3) Double-click on the character to which you want to assign the logo/image
4) Import the SVG file
5) Move the logo/image to center if necessary
6) Close the window and generate the font with File->Generate font
